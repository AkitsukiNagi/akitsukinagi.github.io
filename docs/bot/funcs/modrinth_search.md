本搜尋功能可以透過以下兩種方式使用。

## 1. 簡易式（前綴指令）
指令結構 `!m` `(搜尋類別)` `(搜尋關鍵字)`

* 搜尋類別對應表

| 代號 | 類別 |
|---|---|
| e | 全部（不使用類別） |
| m | 模組 |
| mp | 模組包 |
| r | 資源包 |
| s | 光影包 |
| p | 插件 |

## 2. 互動式（斜線指令）
只需在聊天室中輸入 `/modrinth`，Discord 便會自動引導您完成指令。
